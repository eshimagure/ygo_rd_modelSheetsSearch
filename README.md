# 遊戯王OCG 設定画検索ツール

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

公式X (旧Twitter) で公開された『遊戯王OCG』のモンスター設定画や全身イラストを、カード名・種族・テーマなどで絞り込み検索できる非公式ファンサイトです。

**[👉 実際のサイトを見る](https://eshimagure.github.io/ygo_ocg_modelSheetsSearch/)**

## 主な機能
このツールには、以下の機能が実装されています。

- **イラスト種類の絞り込み**
  - 「すべて」「国内アカウントのみ」「海外アカウントのみ」から表示するイラストの種類を選択できます。（OCGモンスター設定画は海外の公式Xアカウントに更新されている場合があり、このツールではデータを分けて更新・保管してます）

- **キーワード検索**
  - カード名やその読みがな（ひらがな）でインクリメンタルサーチ（入力と同時に検索結果が更新）が可能です。

- **種族・テーマによる絞り込み**
  - 特定の「種族」または「テーマ」にチェックを入れることで、該当するカードを絞り込めます。
  - チェックをすべて外すと、絞り込みが解除され全件表示に戻ります。

- **タブUI**
  - 「種族」と「テーマ」の絞り込みは、分かりやすいタブUIで切り替えることができます。タブを切り替えると、前のタブでの選択は自動的にリセットされます。

- **ページネーション**
  - 検索結果を1ページあたり「30件」「50件」「全件」で表示切り替えが可能です。

- **ソート機能**
  - 検索結果を「更新日付（新しい順/古い順）」「50音順」で並び替えできます。

- **レスポンシブデザイン**
  - PC、タブレット、スマートフォンなど、さまざまなデバイスの画面サイズに対応しています。

## 使用技術

- **スタイリング**: CSS Modules
